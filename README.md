# Task Management System
A simple task management system utilizing user roles and hierarchy.

## Features
- Login/Register
- Admin and User account roles
- First account registered will be assigned as the Super Admin
- Super Admin can manage admin and user accounts

## Tech Stack
- PHP (Vanilla)
- phpMyAdmin
- MySQL
- PDO (Prepared Statements)
- Apache (XAMPP) / Laragon
- Postman (API Testing)

## Architecture
- MVC
- REST
- JSON

## What I learned
- Creating Helpers for general-use functions
- Handling PHP Sessions for account-handling
- Using try-catch statements for error-handling
