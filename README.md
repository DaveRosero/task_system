# Task Management System
A simple task management system utilizing user roles and hierarchy.

## Features
- Login/Register
- Admin and User account roles
- First account registered will be assigned as the Super Admin
- Super Admin can manage admin and user accounts

## Tech Stack
- PHP (Vanilla)
- MySQL
- PDO (Prepared Statements)
- Apache (XAMPP)
- Postman (API Testing)

## Architecture
- MVC
- REST
- JSON
